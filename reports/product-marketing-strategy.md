## 1. 제품 포지셔닝

### 1.1 현재 제품 정의

DreamSync는 **꿈 일기 + AI 분석 + 일상 웰니스 트래킹**을 통합한 하이브리드 웰니스 앱이다. 단순한 꿈 기록 도구를 넘어, 꿈과 일상 감정/컨디션/수면 데이터를 연결하여 **"내일의 컨디션을 예측하는"** 독자적인 가치를 제공한다.

### 1.2 핵심 가치 제안 (Value Proposition)

| 카테고리 | 가치 |
|----------|------|
| **기능적 가치** | 꿈 기록 → AI 심볼/감정 분석 → 컨디션 예보 → 행동 추천 |
| **감정적 가치** | "나를 더 잘 이해하게 되는" 자기 발견의 경험 |
| **사회적 가치** | 무의식과 의식의 연결을 통한 자기 성장 이야기 |

### 1.3 핵심 차별화 요소

1. **예측형 웰니스 (Predictive Wellness)**: 꿈 + 체크인 + 수면 데이터를 융합하여 "오늘의 컨디션 예보"를 생성하는 유일한 앱
2. **30초 체크인**: 저녁 30초 완료를 목표로 한 초경량 일상 기록 (컨디션 → 감정 → 스트레스 → 이벤트)
3. **AI 코치 시스템**: 예보 + 패턴 경보 + 목표 진행도를 종합한 "오늘 코치 플랜" 자동 생성
4. **심볼 사전**: AI가 추출한 꿈 속 심볼의 개인화된 의미 축적 (자기만의 꿈 사전)
5. **UHS (Unconscious Health Score)**: 수면 35% + 스트레스 25% + 꿈 강도 15% + 기분 변동 15% + 예측 오차 10%로 구성된 통합 웰니스 지수
6. **행동 실험 루프**: 추천 행동 실천 → 컨디션 비교 → 피드백, 과학적 자기 실험 프레임

### 1.4 포지셔닝 스테이트먼트

> **DreamSync는** 꿈과 일상의 패턴을 AI로 분석하여 내일의 나를 예측해주는 **예측형 셀프케어 앱**이다.
> 기존 꿈 일기 앱이 "기록"에 머무르는 반면, DreamSync는 기록을 **행동 가능한 인사이트**로 전환한다.

### 1.5 포지셔닝 맵

```
                    높은 AI/분석 역량
                         │
         DreamSync ●     │      ● Calm/Headspace
     (예측+분석+코칭)     │    (명상 중심, 제한적 분석)
```

## 3. 타겟 페르소나

### 3.1 1차 타겟: "의식적 자기 탐구자" (Primary)

**프로필**: 지연 (29세, 여성, 서울 마케터)

| 항목 | 내용 |
|------|------|
| **인구통계** | 25-35세, 도시 거주, 대졸+, 월 300만원+ |
| **심리특성** | 자기 이해 욕구 높음, 명상/요가/저널링 관심, MBTI/에니어그램 등 성격 분석 즐김 |
| **행동패턴** | 매일 아침 일기 쓰는 습관, 꿈을 자주 기억하고 의미를 궁금해함, 웰니스 앱 2-3개 사용 |
| **pain point** | "꿈을 기록하지만 패턴을 스스로 분석하기 어렵다", "내 감정의 원인을 알고 싶다" |
| **gain point** | AI가 꿈과 감정의 패턴을 연결해주고, 내일 대비할 수 있는 행동을 알려준다 |
| **채널** | Instagram, 브런치, 유튜브 자기계발 채널, 팟캐스트 |
| **참고 앱** | Calm, 데이리(감정일기), 포레스트, MBTI 앱 |

|------|------|

| **심리특성** | 데이터 지향적, 수면 품질에 관심, 퍼포먼스 최적화 추구 |

## 8. 차별화 전략

### 8.1 핵심 전략: "예측형 셀프케어" 카테고리 창조

DreamSync의 가장 큰 차별화는 **기록 → 분석 → 예측 → 행동 → 검증**의 완전한 루프를 제공하는 것이다. 기존 앱들은 이 루프의 일부만 제공한다.

```
기존 앱들의 한계:

[꿈 일기 앱]  기록 → 해석       (분석까지만)
[수면 트래킹]  기록 → 통계       (분석까지만)
[웰니스 앱]   기록 → 명상 추천   (일반적 추천)

DreamSync의 루프:

기록(꿈+체크인+수면) → 분석(AI) → 예측(예보) → 행동(코치플랜) → 검증(실험 리포트)
     ↑                                                              │
     └──────────────────── 피드백 루프 ────────────────────────────┘
```

### 8.2 전략적 차별화 기둥 (4개)

#### 기둥 1: 데이터 융합 (Data Fusion)

기존 앱은 단일 데이터 소스만 다룸. DreamSync는 **꿈 + 감정 + 스트레스 + 수면 + 이벤트**를 하나로 통합하여 다차원 분석을 제공한다.

**실행 방안**:
- "다른 앱은 수면 OR 감정, DreamSync는 수면 AND 감정 AND 꿈"을 강조
- UHS 점수의 5가지 구성 요소를 시각적으로 표현

#### 기둥 2: 예측 엔진 (Predictive Engine)

"오늘의 컨디션 예보"는 시장에서 유일한 기능이다.

**실행 방안**:
- 예보 정확도를 지속 공개하여 신뢰 구축 ("평균 72% 정확도")
- 사용자 피드백 루프 (맞았어요/빗나갔어요)로 지속 개선
- "날씨 예보처럼, DreamSync는 당신의 컨디션을 예보합니다" 비유 활용

#### 기둥 3: 개인화 심볼 사전 (Personal Symbol Dictionary)

꿈 해석은 주관적이다. DreamSync는 일반적 해석이 아닌 **개인의 경험에서 축적된 의미**를 가치로 제공한다.

**실행 방안**:
- "물은 모든 사람에게 같은 의미가 아닙니다. 당신만의 심볼 사전을 만들어보세요"
- 심볼 빈도 + 해당 시기 컨디션 상관관계 표시 (Phase 5)
- 심볼 카드를 SNS에 공유할 수 있는 비주얼 생성

#### 기둥 4: 행동 실험 (Behavioral Experiment)

추천 행동을 실천했을 때와 하지 않았을 때의 **실제 컨디션 차이**를 데이터로 보여준다.

**실행 방안**:
- 주간 리포트에 "추천 행동 실천율 vs 컨디션" 상관 그래프
- "실천한 날 평균 컨디션 4.2, 안 한 날 3.1"과 같은 개인 데이터 제시
- 과학적 자기 실험이라는 프레임으로 차별화

### 8.3 방어 가능한 해자 (Moat) 구축

| 해자 | 설명 | 시간에 따른 강화 |
|------|------|----------------|
| **데이터 네트워크** | 사용자가 쌓을수록 예보 정확도 상승 | 강해짐 |
| **개인화된 심볼 사전** | 오래 사용할수록 나만의 데이터 축적 | 전환 비용 높아짐 |
| **습관 형성** | 아침 꿈 기록 + 저녁 체크인 루프 | 리텐션 강화 |
| **한국어 도메인 지식** | 한국 문화 맥락의 꿈 심볼/감정 체계 | 로컬라이제이션 해자 |
